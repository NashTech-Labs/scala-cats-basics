# Scala Cats Basics

- This template is about Scala Cats. 
- It is a library which provides abstraction for functional programming in the Scala programming language.
- This template will help you to understand the basic concepts used in Cats - Type Classes, Implicit Values, Implicit Parameters.

## Prerequisites

- Scala Build Tool(SBT), version 1.4.4
- Scala, version 2.13.4
